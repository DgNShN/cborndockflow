# Support

## How to get help

- Check `README.md` first for setup and usage instructions.
- Review `CONTRIBUTING.md` for development guidance.
- Use GitHub Issues for bugs, requests, or questions about the project.

## Best way to report a problem

Include:

- Your Windows version.
- Python version.
- The exact steps you used.
- The expected result and the actual result.
- Any error message or screenshot.

## What helps most

- A small reproduction case.
- The file type you used.
- Whether the issue happened with OCR, folder watching, or email import.
